# 🚀 求索号考研联盟 (Qiusuo Kaoyan Fleet)

> **项目**: 将个人 Obsidian 考研笔记库升级为代际传承的开源知识母舰
> **舰长**: 第一代舰长（用户）
> **创建日期**: 2026-04-15
> **当前状态**: 🔄 Phase 0 初始化 → 准备进入 Phase 1

---

## 🌌 项目愿景

建立一艘名为「求索号」的考研知识母舰，由第一代舰长（用户）启航。
每一代上岸的学弟学妹作为新一代舰长，基于前代的笔记进行 Fork、优化、拓展，
并通过 Pull Request 将改进反哺母舰，形成 **代际传承的开源知识循环**。

> *「继往开来，一研为定。」*
