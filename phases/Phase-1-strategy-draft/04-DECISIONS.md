# 决策日志

## 2026-04-15 17:20 - 日记公开策略调整

- **决策**: 不再将所有考研日记完全隔离，改为「隐私分级筛查」策略。
- **理由**: 用户认为情绪波动和备考心路历程对学弟学妹具有重要参考价值，完全隐藏会损失珍贵的传承价值。
- **排除**: 「一刀切全部删除」方案。
- **影响**:
  - `5. 考研日记/` 中的日记需要逐篇（或按规则）审查。
  - 涉及个人报考信息、真实姓名电话、具体学校老师等敏感内容的日记必须移出。
  - 纯情绪记录、学习状态起伏、心态调整方法的日记可以保留并公开。
  - README 和 CONTRIBUTING 中需要声明「日记经过脱敏处理」。

## 2026-04-15 17:25 - 考研届数命名规则调整

- **决策**: Release 命名采用「届数制」而非「学年制」。
- **理由**: 用户指出考研行业惯例——25年底考试的学生称为「26届」（26年入学）。
- **排除**: `v2025.1` 这种以考试年份命名的方式。
- **影响**:
  - Release 命名从 `v2025.1` 改为 `v26.1`。
  - README 中的「舰长名录」也需要按届数标注（Gen-1 = 26届）。
  - 文档中所有涉及年份的地方需统一为届数表述。
